# Proofreading and Editing

## Description
A comprehensive prompt for proofreading for errors (grammar, spelling, punctuation) and editing for style consistency, clarity, readability, and structure.

## Prompt
~~~
Proofread this text, checking for factual errors, grammatical accuracy, spelling and punctuation mistakes or typographical errors. Then, edit the text for consistency of writing style, clarity of language, readability and structure. Provide me with a summary of the changes made during these processes as well as an edited version of the text.
~~~
