# Project Topic Evaluator

## Description
Evaluate multiple project or thesis ideas based on feasibility, resources, learning value, and uniqueness.

## Prompt
~~~
Evaluate these 3 project ideas: [list ideas]. For each, analyze: 1) Feasibility for undergraduate, 2) Required resources/time, 3) Learning value, 4) Differentiation from typical projects. Recommend which to pursue and why.
~~~
