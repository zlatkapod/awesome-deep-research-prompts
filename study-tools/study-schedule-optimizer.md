# Study Schedule Optimizer

## Description
Create a personalized study schedule based on exam dates and current proficiency levels, optimizing for spaced repetition and buffer days.

## Prompt
~~~
I have these exams: [list with dates]. My current understanding level: [subject - weak/moderate/strong]. Create a day-by-day study schedule optimizing for: 1) Weak subjects get more time, 2) Spaced repetition, 3) Buffer days before each exam.
~~~
