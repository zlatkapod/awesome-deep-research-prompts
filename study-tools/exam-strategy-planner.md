# Exam Day Strategy Planner

## Description
Create a detailed review plan for the final hours before an exam, optimizing for weak areas and retention.

## Prompt
~~~
I have [X hours] before my [subject] exam. My weak areas: [list]. My strong areas: [list]. Create a minute-by-minute review plan optimizing for: 1) Covering weak areas, 2) Quick strong-area refreshers, 3) Breaks for retention, 4) Sleep time.
~~~
