# Knowledge Graph Builder

## Description
Analyze notes to create a conceptual map showing core and supporting ideas, connections, prerequisites, and applications.

## Prompt
~~~
Analyze these notes on [subject] and create a concept map showing:
1. Core concepts (the big ideas)
2. Supporting concepts (details that explain the core)
3. How concepts connect to each other
4. Prerequisites (what you need to know first)
5. Applications (where this is used)

Format as a text-based hierarchy I can visualize.
~~~
